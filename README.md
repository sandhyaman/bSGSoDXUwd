# 图书馆管理系统 python378

## 项目概述

本项目基于Python和Django框架，实现了一款功能完善的图书馆管理系统。系统集成了爬虫功能，可以从豆瓣网站抓取图书数据，同时支持数据清洗、格式化，并存储至MySQL数据库。此外，系统采用了Django、Bootstrap和Echarts技术栈，提供图书信息管理、借阅记录管理、会员管理以及数据可视化分析等多项功能。

## 技术栈

- Python
- Django
- Bootstrap
- Echarts
- MySQL

## 功能模块

### 图书管理模块

- 图书的增删改查功能
- 图书分类与出版社管理
- 图书库存状态跟踪
- 图书详情查看

### 借阅管理模块

- 创建和管理借阅记录
- 图书归还处理
- 借阅历史查询
- 图书续借功能
- 借阅中心

### 会员管理模块

- 会员信息管理
- 会员卡生成与管理
- 会员借阅历史统计

### 用户认证与权限管理

- 用户登录与注册
- 个人资料管理

### 数据可视化分析

- 借阅统计与趋势图表
- 热门图书分析
- 会员活跃度分析
- 增强版数据分析仪表盘

### 豆瓣图书爬虫

- 从豆瓣爬取图书信息
- 多类别图书数据抓取

## 系统角色

- 系统管理员
- 图书管理员
- 会员
